# スキン画像の配置方法

このディレクトリにゲーム用の画像スキンを配置します。

## ファイル名と対応

| ファイル名 | 用途 |
|---|---|
| `player-blue.png` | プレイヤー1（青）のスキン |
| `player-red.png` | プレイヤー2（赤）のスキン |
| `player-green.png` | プレイヤー3（緑）のスキン |
| `bullet.png` | 弾のスキン |

## 推奨サイズ

- **プレイヤー**: 60 x 60 px（プレイヤー半径 15px × 2 = 30px で描画、2倍解像度推奨）
- **弾**: 20 x 20 px（弾半径 5px × 2 = 10px で描画、2倍解像度推奨）

## 注意事項

- 画像が存在しない、またはロードに失敗した場合は自動的に円形描画にフォールバックします
- PNG（透過あり）を推奨します
- ファイルを配置してブラウザをリロードするだけで反映されます
